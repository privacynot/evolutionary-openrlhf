# 项目简介2

## Evolutionary OpenRLHF 是什么？
Evolutionary OpenRLHF 是一个将 **进化策略** 引入 **大语言模型对齐** 的实验框架。它不再使用固定的奖励模型，而是让一个 **扩散模型** 在种群进化过程中持续学习如何为不同策略打分。整个种群通过“变异‑选择‑对齐”的循环不断提升在目标任务（如数学推理）上的表现。

## 解决的问题
- **奖励稀疏性**：许多复杂任务（如代码生成、数学证明）无法获得密集的奖励信号，只有最终答案是否正确（ORM）。本框架将这种稀疏奖励作为进化压力，驱动种群探索。
- **奖励过拟合**：传统的奖励模型可能被策略模型利用（reward hacking）。通过种群层面的变异与对齐，扩散模型对奖励的评估更加鲁棒。
- **探索效率低**：单策略的 PPO 容易陷入局部最优。种群结合截断扩散的变异算子，能在奖励空间中进行局部探索，同时通过对齐步骤维持策略的稳定性。

## 整体架构
系统包含两个层级的循环：

- **外层（进化循环）**：管理一个由 M 个语言模型策略组成的种群。每一代先训练扩散模型，再选择精英个体，通过变异和对齐产生新个体，最后根据任务完成率（ORM 分数）保留最优种群。
- **内层（个体优化）**：每个个体独立进行一轮 PPO 训练，其奖励信号由当前的扩散模型生成。这一过程完全复用 OpenRLHF 的高效分布式训练管线。

核心组件的交互方式如下：  
1. 扩散模型接收策略模型在推理时产生的隐状态，输出对应的奖励序列。  
2. 变异算子（截断扩散）对精英策略的奖励序列加噪后再去噪，产生探索性奖励。  
3. 对齐步骤用扩散模型的“干净”评估重新校准变异后的策略，防止偏离。
