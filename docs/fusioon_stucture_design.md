# Fusion结构设计

## 1. 输入

模型输入包含两部分：

- τ：Qwen隐藏状态轨迹
- R：扩散模型生成奖励

---

## 2. 为什么需要Fusion

τ 与 R 属于不同语义空间：

- τ：推理过程空间
- R：评价空间

不能直接拼接使用

---

## 3. 编码结构

### τ编码

\[
z_\tau = Enc_\tau(\tau)
\]

---

### R编码

\[
z_R = Enc_R(R)
\]

---

## 4. Fusion方法

### 方法1：拼接

\[
z = [z_\tau ; z_R]
\]

---

### 方法2：注意力融合（推荐）

\[
z = Attention(z_\tau, z_R)
\]

---

## 5. Fusion作用

Fusion学习的是：

> “什么样的思维结构 + 什么样的奖励结构 → 导致高性能模型”
