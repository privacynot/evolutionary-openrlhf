# 方法总览：基于扩散进化的OpenRLHF增强框架

## 1. 问题定义

我们考虑一个三元学习过程：

- τ：Qwen在推理过程中的隐藏状态轨迹（Hidden States）
- R：扩散模型生成的奖励信号（Reward Shaping）
- ρ：最终ORM性能指标（如正确率）

目标是学习一个概率模型：

\[
p(\rho \mid \tau, R)
\]

即：
给定“思维过程 + 奖励结构”，预测该模型最终能达到的性能水平。

---

## 2. 核心思想

本方法的核心不是单纯强化学习，而是三层耦合系统：

### （1）策略层（Qwen / PPO）
学习如何生成回答（policy learning）

### （2）奖励层（Diffusion）
学习“什么样的思维轨迹对应更优奖励结构”

### （3）性能建模层（Beta模型）
学习“训练过程 → 最终性能”的映射关系

---

## 3. 整体流程

1. Qwen生成轨迹 τ
2. diffusion基于 τ 生成奖励 R
3. PPO使用 R 更新策略
4. 得到最终性能 ρ
5. 用 (τ, R) 预测 ρ
6. 反向建模学习“成功路径分布”
