# τ-R-ρ 建模原理

## 1. τ 的定义（思维轨迹）

τ 表示 Qwen 在推理过程中的隐藏状态序列：

\[
\tau = (h_1, h_2, ..., h_T)
\]

它包含：

- 推理步骤结构
- 注意力演化路径
- 中间语义状态

👉 本质：模型“如何思考”

---

## 2. R 的定义（扩散奖励）

R 由 diffusion model 生成：

\[
R = f_\phi(\tau)
\]

含义：

- 根据思维轨迹预测“奖励结构”
- 不是人工标注 reward
- 而是 learned reward shaping

👉 本质：模型“如何被评价”

---

## 3. ρ 的定义（ORM性能）

\[
\rho = \frac{\#correct}{N}
\]

特点：

- 有界变量 [0,1]
- 非高斯分布
- 通常呈现多峰结构（good/bad policy cluster）

👉 本质：模型“最终能力”

---

## 4. 核心建模目标

我们不直接建模：

\[
\tau \rightarrow \rho
\]

而是：

\[
(\tau, R) \rightarrow \rho
\]

原因：

- R 引入“训练过程信息”
- τ 描述“推理结构”
- 两者共同决定最终能力
